Chapitre II

Mesure, Tribu et Mesurabilité

Fondements de la théorie de la mesure

Définition
Tribu (ou σ-algèbre) sur Rⁿ

Ensemble S de parties de Rⁿ vérifiant :

  • Rⁿ ∈ S
  • Stabilité par complémentaire
  • Stabilité par réunion dénombrable
Théorème
Mesure de Lebesgue (λ) - Unicité

Unique mesure vérifiant :

  • Normalisation (volume du pavé)
  • σ-additivité
  • Invariance par translation
Propriété Essentielle
Fonction Mesurable

Sont mesurables :

  • Toute fonction continue
  • Toute fonction continue par morceaux
  • Toute fonction réglée

Stabilité : Limite simple, somme, produit, Sup, Inf de fonctions mesurables sont mesurables

Remarque
Critère de Mesurabilité (Borélienne)

f: R → R est mesurable ssi

∀a ∈ R, f⁻¹(]-∞, a[) ∈ B

où B est la tribu des Boréliens

Chapitre III

Intégrabilité de Lebesgue

Liens avec l'intégrale de Riemann

Définition Clé
Intégrabilité de Lebesgue

f est intégrable ssi :

f mesurable et ∫ |f| dλ < +∞

On définit alors :

∫ f dλ = ∫ f⁺ dλ - ∫ f⁻ dλ
Propriété Fondamentale
Inégalité Triangulaire
|∫ f dλ| ≤ ∫ |f| dλ

Propriété essentielle de l'intégrale de Lebesgue

Théorème
Lien Riemann/Lebesgue (Segments)

Si f est Riemann intégrable sur [a, b] :

∫ f 1[a,b] dλ = ∫ₐᵇ f(x) dx
Théorème
Lien Riemann/Lebesgue (Impropres)

L'intégrale impropre ∫ₐᵇ f(t)dt converge

⟺ Convergence absolue

∫ₐᵇ |f(t)|dt < +∞

⟹ f est intégrable (Lebesgue)

Contre-Exemple Important
Fonctions Non-Intégrables
f(x) = (sin x)/x · 1[0,+∞[

❌ Non intégrable car ∫ |f| dλ = +∞

Même si l'intégrale impropre converge !

Chapitre IV ⭐

Théorème de Convergence Dominée

Priorité Élevée - Corollaires utilisés dans TD 5 et 8

Définition
Négligeable / Presque Partout (p.p.)

Ensemble négligeable :

A négligeable ⟺ λ(A) = ∫ 1ₐ dλ = 0

Propriété vraie p.p. :

Vraie sauf sur un ensemble négligeable

Théorème Clé
TCD - Énoncé

Hypothèses :

  • 1. fₙ → f simplement p.p.
  • 2. Domination : ∃ g intégrable (∫ g dλ < +∞)
    telle que ∀n, |fₙ| ≤ g p.p.

Conclusion :

lim ∫ fₙ dλ = ∫ f dλ
Théorème
TCM (Beppo-Lévi)

Si (fₙ) est une suite croissante de fonctions mesurables positives de limite simple f :

lim ∫ fₙ dλ = ∫ f dλ
Corollaire de Continuité
H(x) = ∫ h(x,t) dλ(t) est continue si :
  • h(x, t) mesurable en t
  • h(x, t) continue en x
  • Domination : |h(x, t)| ≤ g(t)
    avec g intégrable
Corollaire de Dérivabilité
H(x) dérivable et H'(x) = ∫ ∂h/∂x (x,t) dλ(t)

Condition clé :

|∂h/∂x (x, t)| ≤ g(t)

avec g intégrable (domination de la dérivée)

Chapitre V 🔥

Théorèmes de Fubini

PRIORITÉ MAXIMALE

Sujet principal de l'interrogation

Théorème
Fubini-Tonelli (f ≥ 0)

Pour f: R² → R₊ mesurable positive :

∫∫ f dλ = ∫ (∫ f(x,y) dy) dx = ∫ (∫ f(x,y) dx) dy

⚡ Utilisé pour prouver l'intégrabilité

Théorème
Fubini (f intégrable)

Si f: R² → R est intégrable :

∫∫ f dλ = ∫ (∫ f(x,y) dy) dx = ∫ (∫ f(x,y) dx) dy

⚠️ Condition nécessaire :

∫ |f| dλ < +∞
Méthode
Preuve d'Intégrabilité

Appliquer Fubini-Tonelli à |f| :

  1. Calculer ∫ᴿ (∫ᴿ |f(x,y)| dy) dx
  2. Si le résultat est fini
  3. ⟹ f est intégrable ✓
Remarque Importante
Échec de Fubini - Interprétation

Si les deux intégrales itérées donnent des résultats différents

ou si ∫ |f| dλ = +∞

⟹ f n'est pas intégrable

L'égalité des intégrales itérées est la conséquence de l'intégrabilité, non sa cause

Chapitre VI

Changement de Variables en Rⁿ

Technique de calcul en R²

Définition
C¹-Difféomorphisme

φ: U → V est un C¹-difféomorphisme si :

  • φ bijective
  • φ de classe C¹ sur U
  • Jac φ(x) ≠ 0 pour tout x ∈ U
Jacobien = det(Jφ(x))
Théorème
Formule de Changement de Variables
∫ᵥ f(y) dλ(y) = ∫ᵤ f∘φ(x) |Jac φ(x)| dλ(x)
Application Mécanique
Procédure
  1. Remplacer y par φ(x) dans f
  2. Remplacer dλ(y) par |Jac φ(x)| dλ(x)
Remarque
Coordonnées Polaires

Pour φ(r,θ) = (r cos θ, r sin θ) :

|Jac φ(r, θ)| = r

L'élément dλ(x,y) devient :

r dλ(r,θ)
